# 项目介绍

#### 项目名称： MyMovie

#### 简介：

​		个人电影网站，拥有普通用户和管理员以及评论功能

#### 技术：

##### 	前端：

​		html、css、js、bootstrap框架

##### 	后端：

​		采用了SSM框架，Maven自动化构建工具以及MySQL
